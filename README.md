# group-sequential-enrichment-ssr

This repository contains the code to implement the methods in the paper "A Bayesian Group Sequential Enrichment Design with Sample Size Re-estimation".

There are two code scripts in this repository: Design functions AE-SSR.R and Simulation AE-SSR.R.

Design functions AE-SSR.R includes helper functions to implement the methods.

Simulation AE-SSR.R includes code to run simulations.
